# GoodFood

GoodFood startete 2023 als Lebensmittelmanager mit einer damals simpleren,<br/>
aber unübersichlicheren UI.

Nach einigen Verbesserungen der Suchfunktion erschien 2025 GoodFood,<br/>
eine Weiterentwicklung mit neuen Funktionen und einer richtigen UI.

## Funktionen

<li>Lebensmittel eintragen<br/>
<li>Lebensmittel suchen und entfernen<br/>
<li>Lebensmittel anhand der Haltbarkeit finden<br/>
<li>Lebensmittel Räumen zuordnen<br/>
<li>Geteilte GoodFood-Datenbanken

## Installation

Die Installation ist denkbar einfach:<br/>
Finden Sie GoodFood unter [BueroGuide](https://buero-os-toolkit-development.github.io/bueroWebsite#pakete)<br/>
und installieren Sie es einfach mit BueroOS,<br/>
über welches Sie auf der Seite ebenfalls mehr Informationen finden.
